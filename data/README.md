# Data Directory

Structured collections for use with [itemdeck](https://github.com/REPPL/itemdeck/).

## Structure

```
data/
├── collections/
│   └── retro/
│       ├── games/                # My gaming journey
│       │   ├── collection.json
│       │   ├── games/            # 81 games
│       │   └── platforms/        # 13 devices
│       └── commercials/          # Iconic commercials
│           ├── collection.json
│           ├── commercials/      # 13 commercials
│           └── brands/           # 9 brands
└── examples/
    ├── books/                    # 100 best-selling books
    ├── games/vc/                 # 100 best-selling video games
    ├── movies/                   # 100 highest-grossing films
    ├── songs/                    # 100 best-selling songs
    └── tv/                       # 100 highest-rated TV shows
```
